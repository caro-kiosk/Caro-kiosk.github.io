Installation

Basics · Updated 2 hours ago

Install CWiki tools using your preferred package manager. Supports all major platforms.

Prerequisites

  • Node.js 18+ or Python 3.9+
  • Git 2.30+
  • 50MB disk space

Quick Install

Choose your platform:

npm

npm install -g @cwik/tools

pip

pip install cwik-tools

Verify

cwiki --version
cwiki doctor
