Installation

Basics · Updated 2 hours ago

Install CWiki tools and packages using your preferred package manager. The installation process is straightforward and supports all major platforms.

Prerequisites

  • Node.js 18+ or Python 3.9+
  • Git 2.30+
  • 50MB free disk space

Quick Install

Choose your platform and run the appropriate command:

npm (Recommended)

npm install -g @cwik/tools

pip

pip install cwik-tools

Verify Installation

Test that everything works by running: 

cwiki --version
cwiki doctor

Next Steps

Once installed, configure your environment by following the Configuration guide.

setup packages nodejs python
